# Бот для бесед ВК *"Цитаты Аньчоуса"* :cat:

Наипростейший бот для беседы ВК, который присылает текст или картинку, рандомно, по опредленной команде *"Цитата Аньчоуса"*

Моя группа в ВК: [2C Studio](https://vk.com/2cstudio)
